URL Shortener

This application returns a shortened URL that will redirect the user to the original link. If the original URL doesn't follow the the valid http://www.example.com format, the application returns an error.

Instructions:

To shorten a URL, enter the link into this web page's address bar using the /new/ path. The link must be preceded by http:// or https://.

To go to a shortened URL, simply enter it into the address bar. The browser will direct you to the original link.

For example, try the following inputs:

  • https://evening-taiga-49138.herokuapp.com/new/http://www.google.com
  • https://evening-taiga-49138.herokuapp.com/1

Example output:

  • { "original_url": "https://www.google.com", "short_url":"https://evening-taiga-49138.herokuapp.com/1" }

Enjoy!
